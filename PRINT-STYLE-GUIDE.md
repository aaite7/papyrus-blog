# 打印样式使用指南 🖨️

## 功能概述

为 Minimalist Blog 添加了完整的打印优化样式，确保用户在打印文章时获得最佳的纸质阅读体验。

## 特性

### 1. 智能元素隐藏
打印时自动隐藏以下非必要元素：
- 导航栏
- 侧边栏和小部件
- 回到顶部按钮
- 搜索框和筛选标签
- 分享按钮和模态框
- 装饰元素
- 加载动画

### 2. 文章布局优化
- **首页文章列表**: 转换为单列布局
- **文章卡片**: 添加边框，避免分页断开
- **图片**: 限制最大高度，避免溢出
- **字体**: 调整为适合打印的大小 (11pt 正文)

### 3. 单篇文章优化
- **标题**: 24pt 大号字体，居中显示
- **元信息**: 显示日期、阅读量、阅读时长
- **正文**: 11pt 字体，1.8 行高，两端对齐
- **段落**: 首行缩进 2em
- **代码块**: Courier New 字体，带边框和背景
- **引用**: 左侧边框，斜体显示
- **链接**: 显示完整 URL（便于纸质记录）

### 4. 页面设置
- **纸张**: A4 尺寸
- **页边距**: 2cm
- **分页**: 避免在卡片/图片中间断开
- **暗色模式**: 自动切换为白底黑字

### 5. 打印按钮
在文章页 floating bar 添加了打印按钮 🖨️：
- 位置：点赞按钮右侧
- 功能：一键调用浏览器打印对话框
- 提示：悬停显示"打印文章"

## 使用方法

### 方法 1: 点击打印按钮
1. 打开任意文章
2. 点击右下角浮动工具栏的 🖨️ 图标
3. 在打印对话框中设置打印机和选项
4. 点击"打印"

### 方法 2: 浏览器快捷键
- **Windows/Linux**: `Ctrl + P`
- **macOS**: `Cmd + P`

### 方法 3: 浏览器菜单
1. 点击浏览器菜单 (⋮ 或 ☰)
2. 选择"打印"或"Print"

## 技术实现

### CSS 关键样式

```css
@media print {
  /* 隐藏非必要元素 */
  nav, .sidebar, .floating-bar { 
    display: none !important; 
  }
  
  /* 文章分页控制 */
  .manuscript { 
    page-break-inside: avoid; 
  }
  
  /* 图片控制 */
  img { 
    max-width: 100% !important;
    page-break-inside: avoid;
  }
  
  /* 暗色模式适配 */
  body.dark-mode * {
    color: #000 !important;
    background: #fff !important;
  }
  
  /* 链接显示 */
  a[href^="http"]::after { 
    content: " (" attr(href) ")"; 
  }
}

@page {
  margin: 2cm;
  size: A4;
}
```

## 优化建议

### 用户端
1. **打印机选择**: 优先使用激光打印机（文字更清晰）
2. **纸张类型**: 建议使用 80g/m² 或以上纸张
3. **双面打印**: 环保且便于装订
4. **装订边距**: 如需装订，建议在打印机设置中增加左侧边距

### 开发者端
1. **测试打印**: 修改样式后务必测试打印效果
2. **浏览器兼容**: 在不同浏览器中测试（Chrome/Firefox/Safari）
3. **图片优化**: 考虑添加打印专用的低分辨率图片
4. **字体嵌入**: 特殊字体可能需要 embed 才能正确打印

## 已知限制

1. **背景色/图**: 默认浏览器设置为"不打印背景"，用户需手动开启
2. **WebP 图片**: 部分旧打印机可能不支持，建议降级为 JPEG
3. **特殊字体**: 某些系统字体可能无法正确渲染
4. **超长代码**: 超过一页的代码块可能被分页切断
