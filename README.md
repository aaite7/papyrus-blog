# Minimalist Blog

A unique blog platform inspired by ancient manuscripts and scrolls.

## Features

- Ancient manuscript aesthetic with parchment texture
- Handwritten font styling
- Mood tracking for posts
- Automatic reading time calculation
- Like system
- Draft management
- Full Markdown support
- Responsive design

## Tech Stack

- Frontend: Vanilla JavaScript
- Database: Supabase
- Edge Functions: Supabase Functions
- Build Tool: Vite

## Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file with your Supabase credentials:
   ```
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```

## Design Philosophy

Minimalist brings the warmth and elegance of ancient manuscripts to the modern web. With its sepia tones, classic typography, and parchment textures, it creates a reading experience that feels timeless and contemplative.
